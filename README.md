#ManagerFilesTerminall mft
это мой проэкт которий я написал за 1 ноч это файловий менеджер в терминалле написанний на асемблер с помощюу ии суть проэкта сделать работу с файлами еще бистрее где ограниченния ставить только процессор.

#Как Установить  MFT ?

№Как Ползиватса MFT
после установки ви можете запускать прогу если согласилис на алиас то пишите sudo mft для мфт обязателно нужен sudo далше ви попадете в прогу она подерживваеть 2 язика англиский и руский поменять язик можно командой ln
далше после устанвоки язика пришло время узнать все команди h это команда котороя висвитит менеюу хелп где описании все команди также просу бить осторожней с удаленнием веть rd / может 1 кликом удалить вашу ос и данние
поэтому прошу заметить С БОЛШОЙ ВЛАСТЮУ ПРИХОДИТЬ БОЛШАЯ ОТВЕТСТВЕНОСТИ также для копирвианние переместить ми указиваем в формате куда что вот пример  cpf mon/main.py main.py примечанние в куда надо писать
имя того чт ови копириваете также само и чтоб переместить.

№Команди для исползиванния
h смотреть правилний синитексис команд
lf просмотир файлов
nf перейти в фаил или силку
cpf копиривать по напсианиюу что куда
m переместить по написаниюу что куда
cd сооздать директориюу
cf сооздать фаил
ln сменить язик
rf удалить фаил
rd удалить папку
